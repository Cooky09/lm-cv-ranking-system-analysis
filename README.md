# llm-cv-ranking-system-analysis

**AI-powered CV Ranking System**: Automatically parses CVs and job descriptions, extracts relevant skills and experience, calculates similarity scores, and ranks candidates. Uses NLP techniques, LLMs, and keyword analysis to streamline recruitment and improve matching efficiency.

## 🚀 Features
- Parse CVs (PDF, DOCX, DOC) and Job Descriptions
- Extract skills, experience, and relevant keywords
- Compute similarity scores between CVs and job descriptions
- Rank candidates based on matching scores
- Supports visualization and export of results

## 🧰 Tech Stack
- Python 3.10+
- SpaCy for NLP
- Transformers / OpenAI LLMs for enhanced analysis
- Pandas & NumPy for data processing
- FitZ (PyMuPDF) for PDF reading
- Antiword for DOC files
